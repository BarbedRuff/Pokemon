# Pokemon Android app :iphone:
## Описание
Pokemon - это android-приложение разработанное в рамках профильного задания для стажировки в VK.
## Stack
Jetpack Compose, Coroutines, Retrofit2, Coil, <a href="https://pokeapi.co">PokéAPI</a>
